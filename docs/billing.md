# Billing — Clinix

> Estratégia oficial de monetização, cobrança, faturamento e recorrência do Clinix.

Este documento define:

- estratégia comercial
- modelo de cobrança
- planos
- implantação
- recorrência
- formas de pagamento
- inadimplência
- reajustes
- upgrades
- cancelamentos
- limites operacionais
- políticas financeiras
- expansão futura

---

# 1. VISÃO GERAL

O Clinix será posicionado como:

- SaaS premium
- plataforma moderna
- sistema médico profissional
- solução operacional completa para clínicas

O objetivo NÃO é competir por preço.

O objetivo é competir por:

- experiência
- produtividade
- automação
- estabilidade
- suporte
- modernidade
- organização operacional

---

# 2. FILOSOFIA DE MONETIZAÇÃO

O modelo financeiro do Clinix deve priorizar:

- receita recorrente previsível
- margem saudável
- crescimento sustentável
- baixo churn
- clientes de qualidade
- escalabilidade operacional

---

# 3. PRINCÍPIOS FINANCEIROS

O Clinix NÃO deve:

- competir com sistemas baratos
- entrar em guerra de preços
- oferecer recursos ilimitados
- aceitar operações inviáveis financeiramente

---

# O Clinix DEVE:

- cobrar corretamente pelo valor entregue
- proteger margem operacional
- possuir limites claros
- incentivar planos anuais
- monetizar uso avançado
- crescer de forma sustentável

---

| Recurso                   | Limite              |
| ------------------------- | ------------------- |
| Usuários                  | 15                  |

# 18. FORMAS DE PAGAMENTO

O Clinix deve aceitar:

* PIX
* boleto

* transferência
* pagamento anual
* pagamento recorrente

---

# 19. PIX

PIX deve ser incentivado.

---

# Vantagens

* liquidação instantânea
* taxas menores
* menor inadimplência
* automação simples

---

# Estratégia

Oferecer desconto no PIX.

---

# Exemplo

| Forma  | Valor     |
| ------ | --------- |
| Cartão | R$497     |
| PIX    | R$447–477 |

---

# 20. BOLETO

Boleto deve ser suportado.

---

# Regras

* vencimento fixo
* baixa automática
* multa automática
* juros automáticos

---

# Observação

Clientes em boleto possuem maior risco de inadimplência.

---

# 21. CARTÃO DE CRÉDITO

O sistema deve suportar:

* recorrência mensal
* anual antecipado
* parcelamento anual

---

# Gateways recomendados

* Stripe
* Asaas
* Pagar.me
* Mercado Pago

---

# 22. RECORRÊNCIA MENSAL NO CARTÃO

Modelo padrão recomendado.

---

# Exemplo

---

# Vantagens

* previsibilidade

# 23. PLANO ANUAL

O plano anual deve ser fortemente incentivado.

---

# Valor recomendado

```txt
R$4.970/ano
```

---

# Economia aproximada

~2 meses de desconto.

---

# Benefícios do anual

* melhora caixa
* reduz churn
* reduz inadimplência
* aumenta previsibilidade

---

# 24. ANUAL NO CARTÃO

O cliente poderá pagar:

---

# À vista

```txt
1x R$4.970
```

---

# Parcelado

```txt
12x no cartão
```

---

# Estratégia psicológica

O cliente percebe:

```txt
“estou pagando pouco por mês”
```

Mesmo fechando contrato anual.

# 32. POLÍTICA DE INADIMPLÊNCIA

O Clinix deve possuir política clara e automatizada.

---

# Fluxo recomendado

## 1–3 dias

Lembretes automáticos.

---

## 5–10 dias

Bloqueio parcial.

## 15–30 dias

Suspensão operacional.

---

## Após prazo máximo

Encerramento contratual.
